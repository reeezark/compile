# 4.Syntax_Analysis_Top_Down 覆盖清单

- p1-p4：标题、课程提醒、全局思维导图：Top-down parser、Bottom-up parser、RDP、Predictive parser、FIRST/FOLLOW、LL(1)、分析表。
- p5-p8：Parser 类型：自底向上与自顶向下定义、规约/推导关系、手写/工具实现差异、aabb 示例。
- p9-p12：递归下降分析与回溯：每个非终结符对应过程、回溯策略、匹配/失配处理、`x*y` 示例。
- p13-p23：左递归问题与消除：直接/间接左递归、直接左递归通式、多个候选式消除、间接左递归算法、R/Q/S 示例、不同非终结符顺序的影响、RDP 效率问题。
- p24-p29：预测分析与左因子：无回溯递归下降、lookahead、共同前缀问题、左因子提取算法、预测分析的前置条件。
- p30-p50：FIRST 与 FOLLOW：定义、作用、epsilon 处理、计算规则、表达式文法示例及最终 FIRST/FOLLOW 集合。
- p51-p53：LL(1)/LL(k) 文法：三个判定条件、FIRST/FOLLOW 等价表述、LL(1) 不含左递归且无二义、LL(1) 含义、LL(0) 局限。
- p54-p58：LL(1) Parser 实现：递归实现、非递归实现、输入缓冲区、显式栈、预测分析表、非递归算法动作。
- p59-p77：使用预测分析表识别 `id + id * id`：表达式文法、分析表、栈/输入变化、匹配与展开动作、最终 ACCEPT。
- p78-p81：构建 LL(1) 分析表：基于 FIRST/FOLLOW 的填表规则，完整表达式文法分析表。
- p82-p85：判断 LL(1)、非 LL(1) 处理、时间与空间复杂度、LL(k) 表规模。
- p86-p89：总结与延伸阅读。
