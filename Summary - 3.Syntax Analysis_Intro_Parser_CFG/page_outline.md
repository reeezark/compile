# 3.Syntax Analysis_Intro_Parser_CFG 覆盖清单

- p1-p2：课程标题与前置回顾：RE、NFA、DFA、表驱动实现、Thompson、子集构造、DFA 最小化。
- p3-p5：正则语言局限：不能表达计数、括号匹配与嵌套结构；从词法分析转向语法分析。
- p6-p11：编译阶段与语法分析器：输入 token 流、验证 token 串、输出 parse tree，示例区分合法/非法 token 串。
- p12-p16：如何定义语法：自然语言、形式语言、文法四元组、产生式、终结符/非终结符、示例中文文法。
- p17-p23：上下文无关文法 CFG：形式化定义、算术表达式文法、文法简写与符号约定。
- p24-p30：推导：一步/多步推导、句型/句子/语言、从文法生成语言、最左推导与最右推导。
- p31-p41：分析树与二义性：parse tree 定义、yield/frontier、二义文法/二义语言、不可判定性、优先级和结合性消歧。
- p42-p49：乔姆斯基文法体系：0 型无限制文法、1 型上下文有关文法、2 型上下文无关文法、3 型正则文法及比较。
- p50-p55：实践中的语法分析：CFG 的工程价值、parse tree vs AST、课程总结与 Dragon Book 延伸阅读。
